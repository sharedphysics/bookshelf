Issue: "No data loaded" or CORS error

Solution: Your Google Sheet must be published to the web, not just shared.

How to Publish Your Sheet:

  1. Open your Google Sheet
  2. Go to File ‚Üí Share ‚Üí Publish to web
  3. In the dropdown, select the sheet name (e.g., "Sheet1")
  4. Choose format: Comma-separated values (.csv)
  5. Click Publish
  6. Click OK on the confirmation dialog

Alternative: Use Google Sheets API

If publishing doesn't work, you can use the Google Sheets API with a public API key.
